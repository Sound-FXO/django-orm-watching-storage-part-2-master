# Документация к пульту охраны банка

Данный проект позволяет выводить информацию (активность, время входа и выхода, 
длительность посещения) по пропускам сотрудников в браузере в реальном времени.

### Требования к окружению

Python версии 3.14. Остальные требования указаны в `settings.py`

В переменных окружения хранятся переменные `DB_...` с данными для доступа к БД, такие, как
движок БД - `DB_ENGINE`, название БД - `DB_NAME`, имя хоста - `DB_HOST`, порт - `DB_PORT`,
имя пользователя - `DB_USER`, пароль для доступа к БД - `DB_PASSWORD`. 

Также там хранятся разрешенные хосты - `ALLOWED_HOSTS` (`defalut=['127.0.0.1']` - localhost),
а также секретный ключ (главный пароль) - `SECRET_KEY`.

Параметр `DEBUG` предназначен для переключения `debug` режима 
и изначально выключен - `default=False`.

### Запуск 

```bash
$ python manage.py runserver 0.0.0.0:8000
```
Далее в браузере по IP-адресу `127.0.0.1:8000` (localhost) открываем страницу, 
где уже будет запущен наш пульт.
